# The Simplest CycleGAN Full Implementation

- Unpaired Image-to-Image Translation using Cycle-Consistent Adversarial Networks

## Requirement

- TensorFlow >= 2.0.0beta
- TensorLayer >= 2.0.2

## Run

It will automatically download the data in `data.py`.

```python
python train.py
```
